via 春潮频道；主理人序自己动手的快乐，只有自己知道，另外欢迎加入毒奶去广告计划。（里面有丰富的实践示例）重写类...

本文隶属于 实用工具/科学上网 分类，点击分类名称可以查看更多相关文章；

I. 近期更新（重大事件）

08.15.2021 更新 TikTok 不再检测 SIM 卡运营商，只要是海外 IP 就可以使用。所以可不用再搞复杂的代理规则了。 已失效。

II. 如何使用QuantumultX 解锁 TikTok 区域限制

Quantumult X 非官方讨论群组，Tiktok 相关问题可进群咨询，本文解锁复写代码均由群友提供（如花姐），毒奶博客收集整理；

Tiktok version v18.8.5 - v19.4.0 测试可解锁；另，仍请注意当前可解锁的版本IPA文件，并进行备份。以无后顾之忧。

注意不要使用香港节点（代理）！

III. 备份&安装 Tiktok IPA文件

备份以及安装旧版IPA方案

1.iMazing 使用教程（备份IPA/恢复及其他抓包旧版IPA）；

使用imazing备份IPA（当前最新版本）文件免费、方便；博主非常建议大家备份当前可用的Tiktok版本，比如你更新到了（当前最新版本）例如 v19.3.0，刚好是可以解锁的，那么你就应该使用 iMazing 备份 Tiktok（v19.3.0）的IPA文件；（博主已备份v16.6.5 之后所有可以解锁的版本，如若在此之后不小心更新了 Tiktok ，也可以很快的覆盖安装回旧版本（你备份的版本）；）

2.或下载他人提供的IPA文件：https://t.me/Adblock4limbo/1931 （TikTok 17.8.1）并参考：IPhone免越狱安装任意IPA软件教程 安装他人的IPA文件；

下载安装他人的IPA文件，博主未实操，仅供参考；

IV. QuantumultX 共享版本说明

详情说明

*QuantumultX 1.0.3及之后版本增加盗版检测功能（即账号共享），请大家不要使用 共享AppleID 下载的Quantumult X；

进入QuantumltX，点击右下角三菱按钮，点击右上角（或底部的其他设置） ... 更多按钮，滑至底部关于，即可找到设备ID：若出现红色问号会导致 Rewrite (模块)功能不可用），TikTok等解锁不能使用；

V. 解锁Tiktok详细复写配置（v17.8.1-v19.6.0）

本教程中各版本 Tiktok 复写规则详细配置分别来自神机规则（复写）/ Tartarus2014 /Semporia /Orz-3，具体如下：
1.DivineEngine 神机规则；
2.Tartarus2014；
3.Semporia；

如有基础 Github 使用知识的小伙伴可自行查阅，或可参考下方教程进行操作；

VI. 编辑配置文件（教程开始）

*本步骤目前测试支持 QuantumultX v1.0.2 及之后所有版本；

0.分别复制脚本配置文件链接https://raw.githubusercontent.com/NobyDa/Script/master/QuantumultX/Js.conf（NobyDa 去广告重写）

进入QuantumultX，点击页面右下角三菱按钮，找到配置文件-点击编辑-依次找到[rewrite_local]和[mitm]，并在其下方粘贴以下相应代码；

Tiktok 版本 v19.6.0

May 29, 2021 Version 19.6.0

[rewrite_local]
#tiktok 解锁直播区域限制
(?<=_region=)CN(?=&) url 307 JP
(?<=&mcc_mnc=)4 url 307 2
^(https?:\/\/(tnc|dm)[\w-]+\.\w+\.com\/.+)(\?)(.+) url 302  $1$3
(?<=\d\/\?\w{7}_\w{4}=)1[6-9]..(?=.?.?&) url 307 17
[mitm]
hostname = *.tiktokv.com, *.byteoversea.com, *.tik-tokapi.com

Tiktok 版本 18.6.0 - 18.9.5

Apr 8, 2021

v18.8.5-18.9.5 最新版本中注意！

v18.8.5-18.9.5 版本解锁，请把下面复写

(?<=\d\/\?\w{7}_\w{4}=)1[6-9]..(?=.?.?&) url 307 18.4

中的 18.4 改为17即可

(?<=\d\/\?\w{7}_\w{4}=)1[6-9]..(?=.?.?&) url 307 17

最终复写规则为：

[rewrite_local]
#tiktok 解锁直播区域限制
(?<=_region=)CN(?=&) url 307 UK
(?<=&mcc_mnc=)4 url 307 2
^(https?:\/\/(tnc|dm)[\w-]+\.\w+\.com\/.+)(\?)(.+) url 302  $1$4
(?<=\d\/\?\w{7}_\w{4}=)1[6-9]..(?=.?.?&) url 307 18.4

[mitm]

[mitm]
hostname = *.tiktokv.com, *.byteoversea.com, *.tik-tokapi.com

VII. 添加 Tiktok 分流（非必要项）

分流，单独引用 https://raw.githubusercontent.com/DivineEngine/Profiles/master/Quantumult/Filter/StreamingMedia/Video/TikTok.list，开启策略偏好，选择 PROXY 或 其他自定义策略组；

VIII. 生成并信任证书（教程结束）

生成并信任证书

1.进入QuantumultX，点击页面右下角三菱按钮，找到MinM模块，点击生成证书，提示生成成功，点击安装证书此时会跳转至Safari，提示此网站...下载一个配置描述文件。您要允许吗？，点击允许，网页提示已下载描述文件；

2.进入 iOS 系统设置-通用-描述文件-已下载的描述文件-选中，并安装，输入密码...完成描述文件安装；

3.进入 iOS 系统设置-通用-关于本机-证书信任设置-针对根证书启用完全信任-选中刚刚安装的并启用即可；

IX. 开启重写&MITM，享用Tiktok

1.进入QuantumultX，点击页面右下角三菱按钮
2.找到并开启 Rewrite（重写） & MitM 启用按钮；

参阅神机规则-Tiktok复写：https://github.com/DivineEngine/Profiles/tree/master/Quantumult/Rewrite/Unlock

X. QuantumultX Tiktok 换区操作

如想切换到其他地区，进入QuantumultX，点击页面右下角三菱按钮，找到Rewrite模块，点击添加，复制
(?<=_region=)CN(?=&) url 307 UK

将UK修改成其他地区英文缩写即可，其他国家或地区代码。

注意，换区后需继续多刷几个视频或多重启几次才可以发挥效用，不要再问为什么换区不成功了；

XI. FAQ（已知问题及解决办法）

1.不要使用香港节点；
2.不要换区到香港（不要使用区域代码为HK）；
3.建议区域国别代码选择使用UK；
4.尝试清除应用缓存（如果你之前已经成功解锁而后升级版本导致解锁失败，请务必清除应用缓存）；
5.尝试卸载重装；
6.解锁成功后使用iMazing免费备份一下当前可用版本的Tiktok的IPA文件以备下次使用；

    2.进入 iOS 系统设置-通用-描述文件-已下载的描述文件-选中，并安装，输入密码...完成描述文件安装；
