III. Falling snowflakes 介绍

开源、简洁的圣诞节下雪的特效，仅需一段JavaScript代码即可完成部署，支持灵活设置雪花数量、颜色、大小。

开源仓库

JavaScript 对应资源URL：https://unpkg.com/magic-snowflakes/dist/snowflakes.min.js
下雪特效在线测试及效果预览：https://hcodes.github.io/demo-snowflakes/ （可以调试颜色、雪花数量、大小等）

IV. 如何在你的网站上部署

在 </body> 标签前加入如下 JavaScript 代码即可，含<script>标签；

<script src="https://unpkg.com/magic-snowflakes/dist/snowflakes.min.js"></script>
    <script>
        new Snowflakes();
    </script>
